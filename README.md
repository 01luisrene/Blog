## Orden para ejecutar dependencias de gulp

### 1.- Minificar archivos JS
```
gulp minify-js
```

### 2.- Minificar archivos CSS
```
gulp minify-css
```

### 3.- Concatenar archivos JS
```
gulp concat-js
```

### 4.- Concatenar archivos CSS
```
gulp concat-css
```
